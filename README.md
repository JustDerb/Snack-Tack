Snack Tack
==========

Description
-----------
Log in through Facebook and use their college network to find sales around the area.

Tabs
----
### Plan
* Create events for the campus

### Find
* Find events that are not happening now

### Promote
* ‘Create’/tell people about events that are not yours

Functions
---------
Subscribe to the event if you want reminders for it (texting)

Creating event
--------------
 * Name
 * Description
 * Type (Pizza/Ice Cream)
 * Facebook Event
 
Requirements
============

PHP
---
* php_openssl module
* php_mysql module
* php_mysqli module
